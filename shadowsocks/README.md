# shadowsock 一键安装脚本

# 下载安装脚本

`wget --no-check-certificate https://raw.githubusercontent.com/teddysun/shadowsocks_install/master/shadowsocks.sh`

# 1. 执行安装程序

`shadowsocks.sh`


# 2. 开启服务

`/etc/init.d/shadowsocks start`
